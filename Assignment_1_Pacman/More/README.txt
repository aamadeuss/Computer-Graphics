-------------------------
----RUNNING THE GAME-----
-------------------------

Using the html and js file, the game can be launched in a browser. On launch, a map with Pacman and other objects will be displayed.

-------------------------
--------CONTROLS---------
-------------------------

Arrow Keys - Move Pacman in your desired direction.
[/]        - Rotate the map.
(/)*       - Rotate Pacman by 45 degrees in its position.
c          - Change map layout. 3 map layouts available.
m          - Change the user mode. Normal and modify mode available.

*Most keyboards have the (/) keys on the numeric 9/0 keys respectively, therefore the game actually uses the 9/0 inputs.
